# Contributing the Endpointscanner

Thanks for your interest in improving the project! To ensure development and contributions align with the current roadmap milestones, please follow these guidelines:

### Open an issue before a pull request

Before writing any code for a pull request to the repository, **please open an Issue on the repository to discuss your ideas first.**

- Pull requests submitted without a previously discussed and approved Issue will automatically be closed.

### Do not code duplicates

If an idea is already being coded, do not write duplicate code for it. This ensures effort is not wasted.

### What would be appreciated

If you wish to contribute, you may look for open issues (or open one up) regarding:

- Unclear documentation or typos
- Expanding edge-cases for target websites
- Extra flags the community may find useful for the tool.
