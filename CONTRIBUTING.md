# Contributing to EndpointScanner

Thank you for wanting to contribute to this project! There are some rules:

## Rules

### Issues

If you want to make an Issue for EndpointScanner, please make sure:

- It is respectful
- It provides enough details for me to fix

If the Issue lacks details, I may ask you to provide more details like the error message for example.

### Additional Details

Pull requests are not allowed.

Proper punctuation or formatting is not even necessary. I don't care if you put everything in lowercase or make jokes in your Issue. As long as it provides enough details for me to know what to fix and the Issue is understandable, I will work on it.
